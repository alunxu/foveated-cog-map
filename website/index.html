Abstract & Motivation

Holding task and architecture fixed, does the structure of the visual sensor reshape what a navigation agent's memory encodes? We treat this as a controlled silicon case study using a recurrent (LSTM, 3-layer-512) PointNav agent trained with DD-PPO in Habitat on Gibson+MP3D scenes. Only the visual sensor varies across five conditions — from fully blind to uniform high-resolution to foveated with a learned gaze. Linear probing of the LSTM hidden state reveals an encoder–memory race: bottleneck sensors force the recurrent state to carry a strong world-frame GPS code; rich encoders let memory offload spatial integration to the visual route. Representations across conditions occupy linearly disjoint subspaces, and memory transplants are asymmetric — bottleneck-donor states are toxic to rich-encoder recipients. The pattern converges with independent biological evidence across primate, rodent, bat, and blind-human navigation.

H1  ·  Encoder–Memory Race

When the visual encoder supplies little spatial-feature variety, the recurrent state compensates by carrying a world-frame spatial code. Richer encoder output lets memory rely on visual features instead, progressively dissipating its GPS readability.

H2  ·  Format Divergence

Different sensor conditions produce hidden-state representations in distinct linear subspaces, not scaled versions of a common code. Cross-condition memory transplants are asymmetric and direction-dependent.

H3  ·  Gaze & Memory Coupling

Within the rich-encoder regime, where a foveated sensor is centered acts as a second content axis independent of the sensor transform. Learned gaze yields the best overall navigation performance (SPL = 0.82).

Foveation Transform

Our foveation simulates biological vision with eccentricity-dependent blur. Given gaze position (gx, gy):

  1. Compute per-pixel distance from gaze center.
  2. Map distance → eccentricity ∈ [0, 1].
  3. Apply quadratic falloff: σ = eccentricity² × σ_max.
  4. Multi-scale Gaussian blur interpolated from 5 pre-computed levels.

Parameters: fovea_radius=16 px, σ_max=8.0, quadratic falloff. In the learned condition, gaze (x,y) is decoded from the previous LSTM hidden state via MLP(512→64→2)→Sigmoid, training end-to-end.

Probing Setup

  • Freeze agent weights after training; collect LSTM hidden states across episodes.
  • Ridge regression (α = 10) predicting GPS displacement, compass, and absolute position.
  • 5-fold episode-level cross-validation; deterministic rollout.
  • Hewitt–Liang permutation control: shuffled-label R² used as expressivity baseline.
  • Cross-condition probe transfer and linear CKA measure representational geometry divergence.

Results — H1: Encoder–Memory Substitution

Key finding: Visual-encoder capacity tracks top-layer LSTM spatial encoding inversely. Bottleneck conditions (Blind, Coarse) carry a strong linear GPS code (R² 0.78–0.95); rich-encoder conditions (Uniform, Foveated) do not. Cross-training probes confirm the substitution mechanism: rich-encoder agents learn an integrated GPS code transiently (~R² 0.7–0.8 at 50M frames) then progressively dissipate it as the visual route consolidates.

Results — H2: Representational Format Divergence

Key finding: The five agents at comparable task competence (success 93–99%) develop hidden states in linearly disjoint subspaces. Pairwise linear CKA <10⁻⁴ between conditions; 1-NN purity = 1.000 vs. chance 0.20 on pooled hidden states. Cross-condition probe transfer collapses (off-diagonal R² ≪ −800). Memory transplants are asymmetric: bottleneck-donor states are toxic to rich-encoder recipients; the reverse is benign.

Results — H3: Gaze & Memory Coupling

The Foveated (learned) condition achieves SPL = 0.82, the best overall navigation performance across all five conditions, suggesting that active gaze control amplifies the foveation benefit. However, the learned gaze policy exhibits collapse toward a fixed center position during training, which limits the strength of the H3 test. Ongoing experiments — stochastic gaze with bounded-σ Gaussian sampling and multi-seed replication — are designed to restore gaze diversity and provide a clean H3 measurement. The foveated-shifted control (gaze hardcoded at the collapsed position) is queued to isolate static-gaze-location effects from learned-gaze dynamics. Full H3 analysis is pending multi-seed and stochastic-gaze results.
