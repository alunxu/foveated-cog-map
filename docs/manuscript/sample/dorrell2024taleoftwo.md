# A tale of two algorithms: Structured slots explain prefrontal sequence memory and are unified with hippocampal cognitive maps

- **Authors**: William Dorrell, Soraya El-Gaby, Timothy E. J. Behrens, James C. R. Whittington (et al.)
- **Venue**: Neuron
- **Year**: 2024 (epub Nov 21, 2024)
- **DOI**: 10.1016/j.neuron.2024.10.022
- **URL**: https://www.cell.com/neuron/fulltext/S0896-6273(24)00765-7

## Abstract (paraphrased)
Develops a mathematical theory unifying hippocampal episodic memory and prefrontal working memory through a duality between memories stored in synapses (hippocampus) and memories stored in neural activity (PFC). PFC working memory is formalised as controllable activity slots representing dynamic cognitive maps without synaptic plasticity. The theory captures behaviour from list-learning to cue-dependent recall and explains differences, similarities, and trade-offs between hippocampal and prefrontal algorithms.

## Why it is relevant to our paper
Bucket D (cognitive maps / hippocampus theory, recent). Our LSTM h_2 is, conceptually, a *PFC-style activity-slot memory*: it stores the cognitive map in its dynamics, not in long-term weights. Dorrell-Whittington-Behrens give the canonical 2024 mathematical scaffold for that view, explicitly tied to the hippocampal map literature. Pairs with our cross-condition probe transfer and memory-transplant experiments, which are essentially "is the activity-slot code the same code?" tests.
