% ---------- Abstract ----------
\begin{abstract}
Cognitive maps emerge in the recurrent hidden states of navigation agents, as shown for both sighted and blind conditions~\cite{wijmans2023emergence}, but all prior settings keep visual certainty spatially constant, bypassing the need to manage perceptual uncertainty. We introduce a foveated sensor---sharp at the centre, degraded in the periphery---and probe whether the resulting spatial memory merely buffers observations or develops compensatory representations of the agent's own observational uncertainty. We further ask whether foveation drives a shift toward allocentric spatial codes by forcing integration across partial glimpses, and whether learned gaze targets regions where spatial memory is weakest. We train five agents with an identical LSTM backbone on PointGoal navigation in Habitat, varying only visual input (blind, uniform, foveated-fixed, foveated-learned, matched-compute). To investigate the resulting memory structures, we decode spatial variables from frozen hidden states via linear probes, compare representational geometry across conditions with CKA, and identify place-cell-like units through per-unit rate maps.
\end{abstract}


% ---------- Introduction ----------
\section{Introduction}

Wijmans et al.~\cite{wijmans2023emergence} showed that RL agents trained on PointGoal navigation develop map-like representations in their recurrent hidden states---even \emph{blind} agents receiving only goal displacement, GPS, compass, and a close-to-goal scalar. Dwivedi et al.~\cite{dwivedi2022isee} probed sighted agents for related spatial concepts. However, all these settings keep visual certainty spatially constant, leaving open whether the spatial representations are a property of the navigation task or an artifact of uniform input quality.

In biological vision, input quality is never uniform: humans see sharply only within $\sim$2\textdegree{} of gaze. Wirth et al.~\cite{wirth2017gaze} showed that primate hippocampal spatial representations depend on what the animal foveates, not just where it is---suggesting that spatial memory is not merely shaped by where an agent goes, but by how reliably it observes each region. If this holds for learned agents, a foveated sensor should not merely degrade cognitive maps but reshape them: memory may develop qualitatively different strategies to compensate for uneven observation quality.

\subsection{Problem Statement}

These two accounts---passive buffering vs.\ active compensation---make divergent predictions. A passive memory's accuracy should fall with eccentricity, since peripherally-seen regions are stored less faithfully. An uncertainty-aware memory predicts the opposite: accuracy staying flat or \emph{rising}, because the agent devotes representational capacity to uncertain regions. We ask: \textbf{does a foveated agent's memory follow the passive-drop pattern, or does it exhibit the compensatory signature of epistemic representation?}

\subsection{Motivation}

\begin{itemize}
    \item If pure RL under foveation induces uncertainty tracking in recurrent memory, it shows that task pressure alone can discover belief-state-like representations~\cite{hennig2023belief}---without explicit probabilistic objectives or architectural priors for uncertainty.
    \item Foveation degrades visual detail within each frame, preventing reliance on appearance-based position encoding. If this forces more abstract, appearance-invariant spatial representations, it identifies input degradation as a sufficient condition for representational abstraction---an information-bottleneck effect driven by sensor structure rather than architectural choice.
    \item If learned gaze targets memory-weak regions, the navigation objective alone suffices for information-gain-maximizing fixation~\cite{radulescu2022eyemovements}---without auxiliary exploration rewards or hand-designed gaze shaping~\cite{shang2023sugarl}.
\end{itemize}


% ---------- Method ----------
\section{Method}

\subsection{Experimental Setup}

We train all agents on PointGoal navigation in Habitat~\cite{savva2019habitat} using a merged pool of Gibson-0+~\cite{xia2018gibson} (411 scenes) and Matterport3D train (61 scenes), totalling 472 training scenes, following Wijmans et al.~\cite{wijmans2023emergence} Appendix A.1. Evaluation uses 18 held-out MP3D test scenes (1008 episodes), providing an out-of-dataset generalization test.

All agents share an identical 3-layer LSTM (512-d) and the Wijmans non-visual sensor stack: goal-in-start-frame, GPS, compass, and a close-to-goal scalar, each projected to 32-d and concatenated with a 32-d previous-action embedding. Five conditions vary only the visual input (Fig.~\ref{fig:setup}): \emph{blind} (no vision, replicating~\cite{wijmans2023emergence}), \emph{uniform} (full-resolution RGB), \emph{foveated-fixed} (eccentricity-dependent blur with gaze locked at centre), \emph{foveated-learned} (blur centre predicted by a lightweight MLP trained end-to-end), and \emph{matched-compute} (low-resolution uniform RGB exceeding the foveated effective pixel budget, controlling for total information volume).

\subsection{Analysis Framework}

We first confirm task competence: all five agents are evaluated on the 1008 held-out MP3D episodes via Success and SPL~\cite{anderson2018spl}. We then freeze weights, collect per-step LSTM hidden states paired with ground-truth labels (position, heading, eccentricity), and apply four complementary analysis methods~\cite{belinkov2022probing}:
\emph{(i)~Linear probing} with ridge regression from hidden states to spatial targets~\cite{wijmans2023emergence}, validated via control tasks~\cite{hewitt2019control} to ensure probes capture genuine structure.
\emph{(ii)~CKA}~\cite{kornblith2019cka} to compare representation geometry across conditions.
\emph{(iii)~Per-unit rate maps}~\cite{banino2018grid} to identify place-cell-like neurons and test their causal role via ablation.
\emph{(iv)~PCA/t-SNE visualization} of hidden states coloured by position, condition, and eccentricity.

These methods address three hypotheses:

\noindent\textbf{H1 Compensatory memory} Layout-probe accuracy stratified by eccentricity: a passive memory predicts falling accuracy; compensatory memory predicts flat or \emph{rising}. A separate confidence probe (inverse cumulative blur $\sigma$) tests whether input quality is recorded at all.

\noindent\textbf{H2 Representational divergence} CKA between foveated and uniform agents: low similarity despite matched task performance indicates qualitatively different spatial codes. Cross-heading position-probe generalization tests whether foveated codes are more appearance-invariant.

\noindent\textbf{H3 Epistemic gaze} Correlation of learned gaze with position-probe error (memory-uncertainty proxy). If the learned-gaze agent shows stronger H1--H2 effects than the fixed-centre variant, fixation is driven by epistemic demands.

\subsection{Milestones}

\noindent\textbf{M1 (Weeks 1--4):} Training and baseline probing. Four-condition training launched on SCITAS (completed for weeks 1--2). Learned-gaze training, MP3D evaluation, per-unit rate maps.

\noindent\textbf{M2 (Weeks 5--8):} Full H1--H3 analysis across all five conditions. CKA comparison, control-task validation, gaze ablation. Occupancy-map decoding (stretch). Final deliverables: project webpage and presentation.

% ---------- Related Work ----------
\section{Related Work}

\textbf{Cognitive maps in RL agents}\quad Beyond Wijmans et al.~\cite{wijmans2023emergence} and Dwivedi et al.~\cite{dwivedi2022isee} (discussed in Sec.~1), Banino et al.~\cite{banino2018grid} and Cueva \& Wei~\cite{cueva2018grid} found grid-cell-like representations in recurrent networks trained on path integration, and Gornet \& Thomson~\cite{gornet2024cognitive} extended this via visual predictive coding. Hennig et al.~\cite{hennig2023belief} showed that RL under partial observability can induce belief-state-like representations without explicit probabilistic objectives---a precedent for our H1. All these studies use uniform or absent perception; none vary input quality spatially.

\textbf{Foveated vision in deep learning}\quad Mnih et al.~\cite{mnih2014attention} introduced recurrent attention models. Pourrahimi \& Bashivan~\cite{pourrahimi2025foveated} probed a foveated visual-search model for neuroscience-predicted features; Shang \& Ryoo~\cite{shang2023sugarl} jointly learned gaze and motor policies. These address visual search or task performance, not how foveation reshapes a navigation agent's learned spatial memory.

\textbf{Neuroscience and sensor structure}\quad Wirth et al.~\cite{wirth2017gaze} established that primate hippocampal spatial representations depend on gaze, not just location. Atanov et al.~\cite{atanov2024photoreceptor} showed that sensor layout changes downstream task performance. Neither tests whether non-uniform input reshapes learned representations in navigation agents---our central question.

% ---------- Discussion ----------
\section{Discussion}

\subsection{Implications}

Positive results would establish that sensor structure---not just task loss or architecture---is a causal factor in what recurrent memory learns to represent. H1 would show that belief-state-like uncertainty tracking~\cite{hennig2023belief} can emerge from pure RL without probabilistic machinery, suggesting that input degradation itself serves as an implicit regularizer toward epistemic memory. H2 would identify sensor bottlenecks as sufficient for representational abstraction, informing how to design observation pipelines (not just architectures) for embodied agents. H3 would show that task loss alone produces information-gain-maximizing gaze, removing the need for auxiliary exploration objectives. A complete null would bound where the primate foveal--hippocampal link~\cite{wirth2017gaze} breaks down in learned agents.

Mixed outcomes sharpen this picture. H1 without H2: compensatory memory operates within the same coding scheme as uniform vision---sensor structure affects memory \emph{content} but not \emph{geometry}. H2 without H1: foveation reshapes representations without explicit uncertainty tracking---abstraction arises from the integration demand, not from epistemic compensation.

\subsection{Risks and limitations}

\emph{Design scope.} Gaussian blur, deterministic MLP gaze, and a fixed LSTM backbone limit generality but isolate spatial non-uniformity from architectural confounds. To guard against method-specific artifacts, we plan to test alternative foveation models (log-polar transform, ECG image pyramids) and gaze baselines (random gaze) if initial results are promising.

\emph{Learned-gaze collapse.} The gaze MLP could converge to a fixed point; the fixed-centre ablation ensures H1 and H2 remain testable regardless.

\emph{Compute.} Five conditions at 250--500M steps each fit within the SCITAS allocation. The full pipeline (Habitat-sim 0.3.3, DD-PPO, probing) is implemented and tested.
